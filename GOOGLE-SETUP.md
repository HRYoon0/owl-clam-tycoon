# 구글 연동 설정

구글 로그인·드라이브 저장·캘린더 가져오기를 켜려면 **Google Cloud 콘솔에서 OAuth 클라이언트 ID를
한 번 발급**해야 합니다. 코드로는 대신할 수 없는 부분입니다. 10분쯤 걸립니다.

발급한 클라이언트 ID를 `google.js` 맨 위 `CLIENT_ID` 에 붙여넣으면 끝입니다.

---

## 1. 프로젝트 만들기

[console.cloud.google.com](https://console.cloud.google.com) → 상단 프로젝트 선택 → **새 프로젝트**
→ 이름 아무거나(예: `조개는 일을 한다`) → 만들기.

## 2. API 두 개 켜기

**API 및 서비스 → 라이브러리** 에서 검색해 각각 **사용 설정**:

- **Google Drive API** — 게임 저장에 필요
- **Google Calendar API** — 일정 가져오기에 필요

## 3. OAuth 동의 화면

**API 및 서비스 → OAuth 동의 화면**

| 항목 | 값 |
|---|---|
| User Type | **외부** |
| 앱 이름 | 조개는 일을 한다 |
| 사용자 지원 이메일 | 본인 |
| 개발자 연락처 | 본인 |
| 게시 상태 | **테스트** (그대로 둘 것) |

**범위(스코프)** 단계에서 아래 셋을 추가합니다.

```
https://www.googleapis.com/auth/drive.appdata
https://www.googleapis.com/auth/calendar.readonly
https://www.googleapis.com/auth/userinfo.email
```

**테스트 사용자**에 본인 구글 계정을 추가합니다. 다른 사람도 쓰게 하려면 그 계정도 여기에 넣습니다.

> **왜 "테스트"로 두는가**: 캘린더 권한은 구글이 민감 스코프로 분류해서, 앱을 일반 공개하려면
> 검증 심사를 받아야 합니다. 게시 상태를 테스트로 두면 **여기 등록한 사용자 100명까지는 심사 없이**
> 그대로 씁니다. 개인용·학급용이면 이걸로 충분합니다.
>
> 테스트 모드에서는 로그인 시 "확인되지 않은 앱" 경고가 나옵니다. `고급 → 이동`으로 통과합니다.

## 4. 클라이언트 ID 발급

**API 및 서비스 → 사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID**

- 애플리케이션 유형: **웹 애플리케이션**
- 승인된 **JavaScript 원본**에 두 개 등록:

```
https://hryoon0.github.io
http://localhost:8000
```

- 승인된 리디렉션 URI: **비워둡니다** (이 방식에서는 쓰지 않습니다)

만들기를 누르면 **클라이언트 ID**가 나옵니다.
`000000000000-xxxxxxxxxxxxxxxx.apps.googleusercontent.com` 형태입니다.

## 5. 코드에 넣기

`google.js` 첫 줄 근처를 고칩니다.

```js
const CLIENT_ID = "여기에-클라이언트-ID를-붙여넣으세요";
```

---

## 자주 막히는 곳

| 증상 | 원인 |
|---|---|
| `origin_mismatch` 오류 | 4단계 JavaScript 원본에 주소가 빠졌거나 오타. 끝에 `/` 를 붙이면 안 됩니다 |
| `access_denied` | 3단계 테스트 사용자에 그 계정이 없음 |
| 로그인 창이 안 뜸 | 브라우저가 팝업을 차단. 주소창 오른쪽에서 허용 |
| 저장은 되는데 캘린더가 빔 | 2단계에서 Calendar API를 안 켰거나, 오늘 일정이 실제로 없음 |

## 보안 메모

- **클라이언트 ID는 공개돼도 괜찮습니다.** 저장소에 그대로 올라가도 됩니다. 등록된 도메인에서만
  동작하는 것이 보안 장치입니다.
- **클라이언트 시크릿은 필요 없고, 넣어서도 안 됩니다.** 브라우저 앱에 넣으면 누구나 볼 수 있습니다.
  이 방식(토큰 클라이언트)은 시크릿 없이 동작하도록 설계돼 있습니다.
- 게임 저장은 드라이브의 **`appDataFolder`**(앱 전용 숨김 폴더)에 들어갑니다. 사용자 드라이브 목록에
  보이지 않고, 다른 앱은 접근할 수 없으며, 이 앱의 연결을 끊으면 함께 정리됩니다.
- 캘린더는 **읽기 전용**입니다. 이 게임은 일정을 만들거나 고칠 수 없습니다.
