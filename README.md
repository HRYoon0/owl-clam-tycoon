# 조개는 일을 한다

카카오톡 부엉이방에서 시작된 아이디어를 바탕으로 만든 어패류 성장 웹 게임입니다.

할 일을 등록하고 완료하면 경험치와 진주를 얻습니다. 재첩부터 대왕조개까지 진화하고, 진주로 파스타·칼국수·김치 등의 아이템을 모아 비밀 레시피를 발견할 수 있습니다.

## 주요 기능

- 할 일 등록 및 완료 보상
- 7단계 조개 진화
- 요리 아이템 상점과 비밀 레시피
- 브라우저 자동 저장
- 데스크톱·모바일 반응형 화면
- 전용 배경 음악과 효과음
- 설치와 빌드가 필요 없는 정적 웹 게임

## 실행

`index.html`을 브라우저에서 열면 바로 플레이할 수 있습니다.

### 브라우저 위젯

상단의 `▣` 버튼을 누르면 설치 없이 작은 조개 위젯 창이 열립니다.

- Windows와 macOS의 일반 브라우저에서 사용 가능
- 음악 재생·조개 응원·성장 상태 확인
- 메인 게임과 XP·진주·할 일 자동 동기화
- `↗` 버튼으로 전체 게임으로 돌아가기
- 별도 프로그램이나 실행 파일 불필요

브라우저 보안 정책상 위젯 창을 처음 열 때 이 사이트의 팝업 허용이 필요할 수 있습니다. 브라우저가 종료되면 위젯 창도 함께 닫힙니다.

## 기술

HTML, CSS, JavaScript만 사용했습니다.
